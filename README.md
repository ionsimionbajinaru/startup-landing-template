# Startup Landing Template (PulseStack)

A modern, responsive startup landing page template inspired by the visual style of Stripe, Vercel, and Linear.

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript

## Sections Included

- Hero
- Features
- Pricing
- Testimonials
- CTA (Call to Action)
- Footer

## Project Structure

```text
startup-landing-template/
├── index.html
├── css/
│   └── styles.css
└── js/
    └── app.js
```

## Highlights

- Responsive layout for desktop, tablet, and mobile
- Modern SaaS visual language (gradient accents, cards, rounded buttons)
- Reusable UI classes for sections, cards, and buttons
- Clean and maintainable CSS with variables and organized sections
- Mobile navigation toggle with lightweight vanilla JavaScript
- Dynamic footer year rendering via JavaScript

## Getting Started

1. Clone or download this repository.
2. Open `index.html` directly in your browser.

### Optional local server

If you want live reload or cleaner local URL behavior, run a simple local server:

```bash
python3 -m http.server 8000
```

Then open:

- `http://localhost:8000`

## Customization Tips

- Update the brand name in `index.html`.
- Tweak colors in `:root` variables in `css/styles.css`.
- Add/remove pricing tiers or feature cards as needed.
- Connect CTA buttons to your signup flow.
